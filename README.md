# Reading Circle

A personal digital bookshelf built with JavaScript, Node.js, Express and PostgreSQL.

## Features

- 📚 Search books
- 📖 Track reading progress
- ⭐ Rate finished books
- 🔐 Admin mode
- 💾 PostgreSQL persistence
- 📱 Responsive UI

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL

🚧 In Development
